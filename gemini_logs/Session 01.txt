Development Narrative: Session 01
For the initial phase of the project, I collaborated with Gemini 3 Flash to establish a professional development environment on my MacBook Air. We focused on building a modular repository structure that includes dedicated directories for source code, data, and transparency logs, ensuring the project meets high-level software engineering standards for reproducibility. A significant portion of this session involved troubleshooting the acquisition of the COCO 2017 dataset; after encountering repeated failures with standard web downloads, I implemented a terminal-based approach using the wget -c command. This was a deliberate choice to ensure data integrity and allow for resumed downloads of the large 1.2GB files. To finalize the setup, I authored a Python script, src/verify_setup.py, to automate the verification of my local paths and Python environment. Throughout this process, I exercised human oversight by reviewing all AI-suggested terminal commands and adjusting file paths to match my specific macOS directory structure, ensuring the entire pipeline is functional and correctly mapped before beginning the implementation of the deblurring algorithms.
